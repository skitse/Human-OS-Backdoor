# Linguistic Triggers — Language-Level Behavioral Design

> **Domain**: copy systems · messaging psychology · CTA phrasing · prompt timing · bilingual trigger language
> **Use when**: the leverage point is wording, phrasing, tone, cadence, or framing rather than flow structure
> **Scope**: action words, emotion words, identity words, rhythm, contrast, CTA wording, Chinese/English copy cues

---

## Purpose

This file is for language-level intervention, not whole-system design.

Use it when the user's problem sounds like:

- "How should we phrase this?"
- "Why is this copy flat?"
- "Rewrite this CTA / push / headline / hook"
- "Make this more motivating, more trustworthy, or more urgent without becoming manipulative"

Read this file after tiering the task as T1 or T2. For large-system work, use this as a sub-tool rather than the primary playbook.

---

## The Trigger Chain

At the language level, the chain is:

```text
word choice
-> attentional stop
-> emotional coloring
-> self-relevance / identity match
-> perceived effort / risk / reward
-> action or resistance
```

Small wording changes can alter:

- perceived urgency
- trust
- clarity
- agency
- shame / reactance
- felt difficulty

---

## Trigger Families

### 1. Action Triggers

These reduce hesitation by making the next move concrete.

**Best for**:
- CTAs
- onboarding steps
- prompts that need immediate action

**Patterns**:
- start
- get
- see
- try
- unlock
- claim
- continue
- finish

**Examples**:
- "Start your first project"
- "See your sample plan"
- "Continue setup"
- "Get the checklist"

**Avoid**:
- abstract verbs like "optimize", "leverage", "utilize" unless the audience expects them

### 2. Relief Triggers

These lower fear or effort perception.

**Best for**:
- onboarding
- payment reassurance
- sensitive asks

**Patterns**:
- no pressure
- cancel anytime
- takes 2 minutes
- no card required
- preview first
- you can change this later

These are trust-preserving action unlockers.

### 3. Identity Triggers

These increase self-relevance by telling the user "this is for people like me."

**Best for**:
- headlines
- creator content
- niche landing pages
- community prompts

**Patterns**:
- if you're a [specific identity]
- for [specific situation]
- teams like yours
- first-time buyers
- creators who...
- people switching from...

Specific identity beats generic aspiration.

### 4. Emotion Triggers

These add energy, tension, or relief.

**Best for**:
- hooks
- retention copy
- warning copy
- momentum copy

**Positive high-energy**:
- faster
- clearer
- confident
- ready
- ahead

**Negative high-energy**:
- stuck
- missing
- risky
- confusing
- costly

Use negative emotion to name the problem, then move quickly into agency or relief.

### 5. Trust Triggers

These reduce skepticism and make motives legible.

**Patterns**:
- recommended because...
- here's why we ask
- what happens next
- real reviews from...
- verified by...
- backed by...
- no hidden fees
- easy to cancel

Trust language is strongest when specific, not ornamental.

## Rhythm and Cadence

Language affects perceived cognitive load.

### Fast Cadence

Use short clauses when the user already wants to move.

Example:

- "Pick a template. Add your details. Publish when ready."

### Reassurance Cadence

Use short sentence + explanation.

Example:

- "No card required. You can explore the workflow first."

### Tension Cadence

Use contrast and interruption.

Example:

- "Most teams add more tools. The better move is fewer decisions."

### Low-Friction Rule

One sentence should usually carry one psychological job:

- one line stops attention
- one line explains benefit
- one line lowers risk
- one line asks for action

Overloaded sentences reduce trust and clarity.

## Chinese and English Notes

### English Copy Tendencies

English high-conversion copy often rewards:

- concise verbs
- strong subject-action structure
- explicit user benefit
- visible contrast

Examples:

- "Start free"
- "See how it works"
- "Switch in minutes"

### Chinese Copy Tendencies

Chinese copy often carries more force through:

- concise result framing
- stronger implied social meaning
- denser context in fewer characters
- smoother rhythm and paired contrast

Examples:

- "先看效果，再决定"
- "现在开始，随时可停"
- "你不是不会，只是流程太乱"

### Bilingual Caution

Do not translate trigger language literally.

Examples:

- "Claim yours" is often too aggressive when translated directly into Chinese unless the context is promotional and already urgency-framed.
- "No pressure" can feel weak in some English contexts but works well in trust-repair or advisory flows.

Translate the psychological function, not the exact phrase.

## CTA Patterns

### Best Default CTAs

- Start free
- Continue setup
- See your plan
- Try the demo
- Get the guide

### Trust-Sensitive CTAs

- Review before you pay
- See pricing details
- Compare plans
- Talk to an expert
- Preview the workflow

### Re-engagement CTAs

- Pick up where you left off
- Keep your progress
- Finish your setup
- See what's missing

Avoid CTA language that sounds pushy without context:

- "Act now"
- "Don't miss out"
- "Last chance"

Use those only when scarcity is real and already explained.

## Headline Patterns

### Clarity Headlines

- Do [result] without [common pain]
- A simpler way to [task]
- The fastest way to [specific outcome]

### Trust Headlines

- Clear pricing. No hidden surprises.
- Built for first-time buyers who want to check everything first.
- See exactly how the process works before you commit.

### Identity Headlines

- For teams that need speed without chaos
- For creators tired of posting into the void
- For first-time founders who want a cleaner start

### Contrast Headlines

- Less setup. More signal.
- Fewer steps. More confidence.
- Less guesswork. Better decisions.

---

## Trigger Selection Guide

If the user is:

- **hesitating** -> use action + relief
- **skeptical** -> use trust + specificity
- **unmotivated** -> use identity + progress
- **overwhelmed** -> use clarity + low-friction cadence
- **scrolling past** -> use contrast + curiosity

---

## Failure Modes

Language backfires when:

- urgency appears before legitimacy
- identity language is too broad or fake
- emotional language outruns the actual value
- trust language is generic ("trusted by many") instead of concrete
- copy sounds translated instead of native to the channel

Common mistakes:

- too many adjectives, not enough verbs
- sounding persuasive before sounding clear
- using hype language in trust-sensitive categories
- using shame as motivation when reassurance would work better

---

## Safe Use Rules

- Prefer real clarity over pressure
- Prefer self-efficacy over guilt
- Prefer transparent recommendation over covert manipulation
- In high-stakes contexts, use reassurance and explanation before urgency
